Text and Lists Definitions

Chapter 2 - Text

Headings

<h1>
main headings
<h2>
subheadings
<h3>
further sections under subheadings, so on and so forth
<h4>
further sections under subheadings, so on and so forth
<h5>
further sections under subheadings, so on and so forth
<h6>
further sections under subheadings, so on and so forth

Paragraphs

<p>
By default, a browser will show each paragraph on a new line with some space between it and any subsequent paragraphs.

Bold & Italic

<b>
Bold - represents a section of text that would be presented in a visually different way (for example keywords in a paragraph). Its use does not imply any additional meaning.
<i>
Italic - represents a section of text that would be said in a different way from surrounding content -- such as technical terms, names of ships, foreign words, thoughts, or other items that would usually be italicized.

Superscript and Subscript

<sup>
Used to contain characters that should be superscript such as the suffixes of dates or mathematical concepts like raising a number to a power such as 22.
<sub>
Used to contain characters that should be subscript. It is commonly used with foot notes or chemical formulas such as H2O.

Line Breaks and Horizontal Rules

<br />
Adds a line break.
<hr />
Used to create a break between themes.

Strong and Emphasis

<strong>
Indicates the element has strong importance. By default, browsers will show contents in bold.
<em>
Indicates emphasis that subtly changes the meaning of a sentence. By default, browsers will show contents in italic.

Quotations

<blockquote>
Used for longer quotes that take up an entire paragraph. the <p> element is still used inside the <blockquote> element. Browsers tend to indent contents.
<q>
Used for shorter quotes within a paragraph. Browsers are supposed to put quotes around the <q> element however Internet Explorer does not -- therefore many people avoid using the <q> element.

Abbreviations & Acronyms

<abbr>
Used for abbreviations or acronyms. A title attribute on the openeing tag is used to specify the full term.

Citations & Definitions

<cite>
Used when referencing a piece of work such as a book film or research paper, can be used to indicate where the citation is from.
<dfn>
Used the first time you explain some new terminology (perhaps an academic concept or some jargon) in a document, it is known as the defining instance of it.

Author Details

<address>
Used to contain contact details for the author of the page. It can contain a physical address, but it does not have to. For example, it may also contain a phone number or email address.

Changes to Content

<ins>
Used to show content that has bee inserted into a document.
<del>
Used to show text that has been deleted from a document.

Chapter 3 - Lists

Ordered Lists

<ol>
Lists where each item in the list is numbered. For example, the list might be a set of steps for a recipe that must be performed in order, or a legal contract where each point needs to be identified by a section number.

Unordered Lists

<ul>
Lists that begin with a bullet point (rather than characters that indicate order).

List Items

<li>
Placed around each item within a list.

Definition Lists

<dl>
Made up of a set of terms along with the definitions for each of those terms.
<dt>
Designates the definition term (the term being defined).
<dd>
Contains the definition for the term declared.

Nested Lists

<ul>, <ol>, <dl>
A second list can be placed within a list item to create a nested list.
